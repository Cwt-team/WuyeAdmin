# 物业管理系统 (前端R)

这是物业管理系统的新版前端项目，基于Vue 3、Element Plus和Vue Router构建。

## 项目设置
```
npm install
```

### 开发模式
```
npm run serve
```

### 编译和打包
```
npm run build
```

### 代码格式检查
```
npm run lint
```

## 项目结构

- `src/` - 源代码目录
  - `assets/` - 静态资源
  - `components/` - 通用组件
  - `router/` - 路由配置
  - `utils/` - 工具函数
  - `views/` - 页面组件
  - `App.vue` - 根组件
  - `main.js` - 入口文件
- `public/` - 公共静态资源目录
- `docs/` - 项目文档
  - [开发总结](./docs/development-summary.md) - 项目开发总结文档

## 技术栈

- Vue 3
- Vue Router
- Element Plus
- Axios
- ES6/7
